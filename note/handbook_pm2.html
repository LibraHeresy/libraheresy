命令 ​

pm2 start（启动） ​

shell
# 启动应用
pm2 start app.js

# 启动应用并监听文件改动
pm2 start app.js --watch
# 启动应用
pm2 start app.js

# 启动应用并监听文件改动
pm2 start app.js --watch

pm2 restart（重启） ​

shell
# 重启指定应用
pm2 restart app

# 重启所有应用
pm2 restart all

# 重启多个应用
pm2 restart app1 app2 app3
# 重启指定应用
pm2 restart app

# 重启所有应用
pm2 restart all

# 重启多个应用
pm2 restart app1 app2 app3

pm2 link（链接官方在线监控） ​

  1. 访问 pm2.io

  2. 执行 pm2 link XXX XXX，关联服务

  3. 执行 pm2 plus，自动打开在线监控网页
